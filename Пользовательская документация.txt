

Пользовательская документация Hardload.exe

Программа Hardload.exe предназначена для копирования файлов.
Пользователь при открытии программы видит перед собой интерфейс для выбора файлов для копирования, они отображаются в окне программы. Ниже пользователь может выбрать папку в которую следует скопировать выбранные файлы. Программа поддерживает все основные форматы файла, позволяет выбрать файлы для копирования из разных папок. 

Функции программы:

- Копирование файлов в выбранную папку.

Алгоритм работы программы:

- Выбрать файлы (с помощью кнопки выбрать файл)
- Добавить файл в список для копирования файлов
- Выбрать папку куда скопировать файлы
- Начать копирование (с помощью кнопки начать копирование)
